## üí° Use Case

The application performs the following:

1. Accepts a string input from the user.
2. Reverses the input string.
3. Compares the original string with the reversed string.
4. Displays:

    * "It is a Palindrome" if both are equal.
    * "It is Not a Palindrome" otherwise.
